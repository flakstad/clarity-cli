# Clarity docs (CLI)

For command reference:
- `clarity --help`
- `clarity <command> --help`

For long-form docs:
- `clarity docs` (list topics)
- `clarity docs <topic>` (show topic)

Recommended starting topics:
- `getting-started`
- `output-contract`
- `identity-ownership`
- `items-outlines`
- `deps`
